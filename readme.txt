== Description ==

The Taskums API WordPress Plugin. Connect to your Taskums.com account from WordPress and display stats!

Create an account on http://taskums.com and find your API Key on the My Account page.

**Features**

* Connects to your Taskums account using the API
* Provides a Quick API for common needs
* Provides a Full API for complete control of the output
* Easy to edit template files for Quick API
* Plenty of code examples for both Quick and Full APIs

The taskums plugin is only for use with taskums.com accounts. Go to http://taskums.com for more information on this service.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `taskums` plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add you API Key in the Taskums Settings
4. Follow Usage instructions to use in your theme
